# PicPaste

智能剪贴板图片处理工具 - 截图自动转文件路径

## 开发者
潇洒公子

## 功能特点

- **截图自动处理**：检测到剪贴板中的截图后，自动保存为 PNG 文件
- **智能替换**：将剪贴板中的图片替换为文件路径，方便在终端/命令行中粘贴使用
- **自定义保存路径**：可设置截图保存的文件夹位置
- **缓存管理**：
  - 最大文件数量限制
  - 文件保留时长设置
  - 自动清理间隔
  - 退出/关机时自动清理
- **开机自启动**：支持 Windows 开机自动运行
- **日志记录**：内置日志查看功能，方便排查问题

## 使用方法

1. 运行 `PicPaste.exe`
2. 截图（使用任意截图工具）
3. 在终端/命令行中按 `Ctrl+V` 粘贴，即可得到图片文件路径

## 设置说明

右键点击托盘图标 → 设置，可以配置：

- **保存路径**：截图文件保存的位置
- **处理模式**：
  - 仅在终端粘贴时处理（推荐）
  - 截图后立即处理
- **缓存管理**：
  - 最大文件数量（0 = 无限制）
  - 保留时长（0 = 永久保留）
  - 清理间隔（0 = 不自动清理）
- **启动设置**：开机自动启动
- **清理选项**：退出/关机时自动清理截图文件

## 系统要求

- Windows 10/11 64位系统
- 无需安装 .NET Runtime（已自包含）
